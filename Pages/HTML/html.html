      Introduction to HTML

      HTML is an abbreviation of HyperText Markup Language. It's a set of instruction given to web browsers to render the pages. It's often used a skeleton of the website. It's used to determine what text, images, sections is included in the website page. It's also used to organize the contents.

      HTML Files

      All html files have the extension of .html. You can create them by creating a text file and then change .txt extension to .html extension. You can also start by opening a text editor or an ide and then save the files as an html file.

      Setting up HTML files

      There are some necessary tags that needs to be put in every html files. html tag head tag body tag
